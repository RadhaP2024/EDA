# 📊 Sales Data - Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on a sales dataset using **Python**, **Pandas**, and **Matplotlib**. The objective is to understand the dataset by examining its structure, identifying missing values, detecting duplicate records, generating descriptive statistics, and visualizing data distributions.

---

## 🚀 Features

- Load the sales dataset
- Display first and last five records
- View dataset information
- Check dataset shape
- Display column names and data types
- Find missing values
- Detect duplicate records
- Generate descriptive statistics
- Compute correlation matrix
- Plot histograms for numerical columns
- Generate box plots
- Display correlation heatmap
- Show value counts for categorical columns
- Print dataset summary
